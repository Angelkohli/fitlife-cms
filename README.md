# FitLife Winnipeg Fitness Center CMS

A content management system for managing fitness class schedules, instructors, and member reviews.

## Tech Stack
- HTML
- PHP
- MySQL with PDO
- Apache Server
- CSS
- JavaScript

## Features (In Progress)
- [ ] Admin class management (CRUD)

- [ ] Comment & review system
- [ ] Image uploads
- [ ] Search functionality

## Database Setup
1. Import `database/schema.sql` to create tables
2. Import `database/seed.sql` to populate with sample data
3. Update `config/database.php` with your credentials

## Local Development
1. Place project in `htdocs` (XAMPP) or `www` (WAMP)
2. Start Apache and MySQL
3. Navigate to `http://localhost/fitlife-cms/`
